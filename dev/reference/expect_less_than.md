# Deprecated numeric comparison functions

These functions have been deprecated in favour of the more concise
[`expect_gt()`](https://testthat.r-lib.org/dev/reference/comparison-expectations.md)
and
[`expect_lt()`](https://testthat.r-lib.org/dev/reference/comparison-expectations.md).

## Usage

``` r
expect_less_than(...)

expect_more_than(...)
```

## Arguments

- ...:

  All arguments passed on to
  [`expect_lt()`](https://testthat.r-lib.org/dev/reference/comparison-expectations.md)/[`expect_gt()`](https://testthat.r-lib.org/dev/reference/comparison-expectations.md).

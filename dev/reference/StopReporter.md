# Error if any test fails

The default reporter used when
[`expect_that()`](https://testthat.r-lib.org/dev/reference/expect_that.md)
is run interactively. It responds by displaying a summary of the number
of successes and failures and
[`stop()`](https://rdrr.io/r/base/stop.html)ping on if there are any
failures.
